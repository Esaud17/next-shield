isAuth

isAuth: boolean
  • Defined in next-shield/NextShield.tsx:33

🔑 This value must be provided by the state of your app. Indicates if the user is authenticated or not.

example

Here is a simple example with firebase auth. But applies the same logic for any auth provider. 😋

example
const [isAuth, setAuth] = useState(false)
useEffect(() => {
 const unsubscribe = auth().onAuthStateChanged(user => {
   if (user) {
      setAuth(true)
      return
    }

   setAuth(false)
  })

  return () => unsubscribe()
}, [isAuth])

isLoading

isLoading: boolean
  • Defined in next-shield/NextShield.tsx:59

⏳ This value must be provided by the state of your app. Indicates if the user is already available or not.

example

Here is a simple example with firebase auth. But applies the same logic for any auth provider. 😋

example
const [isAuth, setAuth] = useState(false)
const [isLoading, setLoading] = useState(true)
useEffect(() => {
  const unsubscribe = auth().onAuthStateChanged(user => {
    if (user) {
      setAuth(true)
      setLoading(false)
      return
    }

    setAuth(false)
    setLoading(false)
  })

  return () => unsubscribe()
}, [isAuth])
